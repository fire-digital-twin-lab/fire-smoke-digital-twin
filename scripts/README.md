# Scripts

- `bootstrap_repo.cmd`: tạo/điền scaffold đầy đủ bằng Python, không ghi đè file không rỗng nếu không có `--force`.
- `migrate_current_layout.cmd`: di chuyển layout hiện tại vào namespace rồi chạy bootstrap.
- `create_tree_only.cmd`: CMD thuần, chỉ tạo cây thư mục và placeholder.
- `check.cmd`: compile + ruff nếu đã cài.
- `test.cmd`: pytest mặc định, bỏ external/slow.
